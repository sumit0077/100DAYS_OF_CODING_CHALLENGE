Task
Given an n integer,

, perform the following conditional actions:

If n is odd, print Weird
If n is even and in the inclusive range of 2to5, print Not Weird
If n is even and in the inclusive range of 6to20 , print Weird
If n is even and greater than 20, print Not Weird
Complete the stub code provided in your editor to print whether or not
is weird.



Sample Input 0

3

Sample Output 0

Weird

Sample Input 1

24

Sample Output 1

Not Weird
